# Metaverse Dakwah - Sidang Skripsi Zaki Mubarok

Aplikasi presentasi interaktif untuk sidang skripsi tentang penggunaan Virtual Reality dalam komunikasi dakwah modern.

## Fitur Utama

- 📊 **Presentasi Interaktif**: Navigasi slide dengan kontrol penuh
- 📱 **Responsive Design**: Optimal di perangkat desktop dan mobile
- ⌨️ **Keyboard Navigation**: Arrow keys dan Space untuk navigasi
- 🎨 **Theme Mewah**: Gold dan dark theme yang elegan
- 📤 **Share Links**: Bagikan slide spesifik dengan URL
- 🔄 **Autoplay**: Mode putar otomatis dengan kecepatan dapat disesuaikan
- 🖥️ **Fullscreen Mode**: Presentasi layar penuh

## Tech Stack

- **Frontend**: React 19 + TypeScript
- **Styling**: Tailwind CSS 4 dengan @tailwindcss/vite
- **Animation**: Motion (Framer Motion alternative)
- **Build Tool**: Vite 6
- **Icons**: Lucide React

## Installation

```bash
npm install
```

## Development

```bash
npm run dev
```

Server akan berjalan di http://localhost:3000

## Build

```bash
npm run build
```

## Preview

```bash
npm run preview
```
